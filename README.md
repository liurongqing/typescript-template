# typescript-template

typescript 模板包括 Eslint + Prettier

## 快速学习

1. 安装
    ```bash
    npm install
    ```

2. 运行
    ```bash
    npm run build # ts 编译成 js
    npm run start # 运行 js
    ```


## VSCode 准备工作

安装 eslint
安装 prettier


node 最好 18+ 以上， 这样才支持 `--watch`

```bash
node --watch filepath 
```
